# Perpustakaan
Laravel - Daftar Anggota Perpustakaan (Tailwind css)

### Run Aplikasi
- php artisan migrate
- php artisan db:seed --class=AnggotaSeeder
- php artisan serve & npm run dev

## Fitur Aplikasi
- Melihat index data master anggota
- Menambah anggota
- Melihat detail anggota
- Mengubah data anggota
- Menghapus anggota
